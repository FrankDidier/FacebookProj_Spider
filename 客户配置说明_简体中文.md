# 📋 客户配置说明 - 超简单版

## ❓ 您的疑问解答

### 问题 1: "fbapi 密钥怎么配置？"

**回答**: **没有 fbapi 密钥！** 您可能混淆了两个概念：

1. **浏览器 API 密钥** (这个才是需要的)
   - AdsPower 需要 API 密钥
   - BitBrowser **不需要** API 密钥
   
2. **Facebook API 密钥** (不需要！)
   - 我们的软件**不需要** Facebook API 密钥
   - 我们通过浏览器操作 Facebook，不是通过 Facebook API

---

### 问题 2: "这个本地demo 我这边需要下载吗？"

**回答**: **不需要下载！** "本地 demo" 是指：

- BitBrowser 自带的本地服务模式
- 当您启动 BitBrowser 后，它会自动在 `http://127.0.0.1:54345` 提供服务
- 我们的软件会自动连接到这个地址
- **您只需要下载并安装 BitBrowser 浏览器本身**

---

### 问题 3: "你需要你那边先调试第一部先跑通一下"

**回答**: 我理解您的顾虑。让我给您完整的配置步骤。

## 🚀 完整配置步骤（BitBrowser 版本）

### 第一步：下载并安装 BitBrowser

1. **下载 BitBrowser**
   - 官网：https://www.bitbrowser.cn
   - 下载适合您系统的版本（Windows/Mac）
   - 安装到您的电脑

2. **启动 BitBrowser**
   - 安装完成后，启动 BitBrowser
   - 确保它正在运行
   - 它会自动在后台提供服务（端口 54345）

### 第二步：在 BitBrowser 中添加 Facebook 账户

1. **打开 BitBrowser**
2. **创建新的浏览器环境**
   - 点击 "新建浏览器" 或类似按钮
   - 选择 "Facebook" 平台
3. **登录 Facebook**
   - 在新创建的浏览器环境中登录您的 Facebook 账户
   - 完成任何验证步骤
4. **重复添加更多账户**（可选）
   - 建议添加 2-3 个 Facebook 账户
   - 每个账户在单独的浏览器环境中

### 第三步：配置我们的软件

1. **打开我们的软件（FacebookMarketingTool.exe）**

2. **进入「配置向导」**
   - 点击左侧第一个标签：⚙️ 配置向导

3. **选择浏览器类型**
   - 在 "浏览器类型" 下拉框中选择：**BitBrowser**
   
4. **查看 API 密钥字段**
   - 您会看到 API 密钥输入框变成灰色（禁用）
   - 显示文字：**"BitBrowser 不需要 API 密钥"**
   - 这是正常的！**不需要填写任何东西**

5. **（可选）设置浏览器路径**
   - 如果 BitBrowser 已经在运行，这一步可以跳过
   - 如果要填写，路径类似：`C:/Program Files/BitBrowser/BitBrowser.exe`

6. **设置账户数量**
   - 输入您在 BitBrowser 中添加的 Facebook 账户数量
   - 例如：如果添加了 3 个账户，就填 `3`

7. **点击「保存配置」**

8. **点击「重新验证」**
   - 等待验证完成
   - 如果看到 ✅ 绿色勾，说明配置成功

### 第四步：开始使用

验证成功后，您就可以使用所有功能了：
- 采集群组
- 采集成员
- 自动点赞
- 自动评论
- 自动私信
- ... 所有功能

## 🔍 如果验证失败

### 错误：BitBrowser 服务未检测到

**解决方法**：
1. 确认 BitBrowser 正在运行
2. 在任务管理器中查看 BitBrowser 进程
3. 重启 BitBrowser
4. 在我们的软件中点击「重新验证」

### 错误：未找到 Facebook 账户

**解决方法**：
1. 在 BitBrowser 中添加 Facebook 账户
2. 确保账户已登录 Facebook
3. 在我们的软件中点击「重新验证」
